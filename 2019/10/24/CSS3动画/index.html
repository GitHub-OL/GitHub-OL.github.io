CSS3动画

2019-10-24

一.圆角

  • 语法: border-radius:左上 右上 右下 左下

二.阴影

盒阴影

  • 语法: box-shadow:h-shadow v-shadow blur spread color inset (水平阴影 垂直阴影 模糊距离 阴影尺寸 阴影颜色 外部阴影改为内部阴影)

文字阴影

  • 语法: text-shadow:h-shadow v-shadow blur color (水平阴影 垂直阴影 模糊距离 阴影颜色)

三.渐变(gradients)

线性渐变

  • 语法: background: linear-gradient(direction, color-stop1, color-stop2, …)

径向渐变

  • 语法: background: radial-gradient(center, shape(circle/ellipse)， size, start-color, …, last-color)

四.转换

2D转换

  • 语法: transform:translate() rotate() scale() skew() (转换 旋转 缩放 倾斜)

3D转换

  • 语法: transform:translate3d() rotate3d() scale3d() perspective()

五.过渡

  • 过度属性:transition-property:none | all | property
  • 过度时间:transition-duration:s | ms
  • 过度函数:transition-timing-function:ease | linear | ease-in | ease-out | ease-in-out
  • 过度延迟:transition-delay:s | ms
  • 简写语法: transition:property duration timing-function delay;

六.动画

  • @keyframes(关键帧)

  • animation(动画属性)

七.媒体查询

  • 语法: @media mediaType and|not|only (media feature){ //css code }

八.多列布局(multi-column)

  • 列数:column-count
  • 列间距:column-gap
  • 列分割线样式:column-rule
